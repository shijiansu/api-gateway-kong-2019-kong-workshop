Lab Gateway 04 (Rate Limiting)

We want to limit the calls to an endpoint by adding a rate limiting

  1. Create a new route based on the service from lab 1 with the path /rateLimiting
  2. Add a route based rate-limiting of 60 calls per hours, and 3 per minute
  3. Add key-auth and two new consumers
  4. Make an exception for one of your previously added Consumers and allow 600 calls per hour.
