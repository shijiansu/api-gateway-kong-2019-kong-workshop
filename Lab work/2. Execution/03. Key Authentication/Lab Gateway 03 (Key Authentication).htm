Lab Gateway 03 (Key Authentication)

Add authentication to your API

  1. Create a new route based on the service from lab 1 with the path /keyAuth
  2. Add the key-auth plugin to the service you created. Check access being blocked after plugin has been enabled.
  3. Regain access to the service again by adding a Consumer and key-auth credentials
